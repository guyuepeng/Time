## 功能

- 主页显示从纪念日开始计时，距今时长。可以以多种模式（月、周、日、时）显示
- 用户可随时更新首页背景
- 可以下载高清背景图到本地
- 首页显示当日天气状况及气温
- 应用内自动更新
